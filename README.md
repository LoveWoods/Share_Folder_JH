💡 핵심 개념 요약

⬆️ 푸시 = 클라우드에 업로드
⬇️ 풀 = 클라우드에서 다운로드
✅ PR = "내가 바꾼 거 승인해줘" 요청

🍴 포크 = 남의 저장소를 내 저장소로 복사
📥 클론 = 원격 저장소를 내 PC로 다운로드 (연결고리 유지)
